\doxysubsection{Detailed Description}
Manages order books for different trading symbols. 

This class is responsible for\+:
\begin{DoxyItemize}
\item Processing and storing order book data from the exchange
\item Maintaining up-\/to-\/date local copies of order books
\item Providing access to the latest order book state for each symbol
\item Thread-\/safe access to order book data 
\end{DoxyItemize}
